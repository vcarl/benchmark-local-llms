### 2.2 Sliding Window Attention

We introduce a new sliding window attention mechanism to reduce the computational cost of attention. Traditional self-attention has a computational complexity of $O(n^{2})$, which can be prohibitive for long sequences. Our sliding window attention mechanism reduces this complexity to $O(n\log n)$.

The sliding window attention mechanism works by dividing the sequence into windows of a fixed size. The model only attends to tokens within the same window, which significantly reduces the number of attention operations. However, this can lead to a loss of information, as tokens in different windows are not directly connected. To mitigate this, we use a sliding window approach, where the windows are shifted by a certain number of tokens. This allows the model to gradually build connections between tokens in different windows.

Mathematically, the sliding window attention can be defined as follows. Let $X$ be the input sequence of length $n$, and let $w$ be the window size. We divide $X$ into $k=\lceil\frac{n}{w}\rceil$ windows of size $w$ (with the last window possibly being smaller). For each window $i$, we define the attention mask as:

\[ a_{ij}=\begin{cases}1&\text{if }|i-j|\leq 1\\
0&\text{otherwise}\end{cases} \]

This means that each window only attends to itself and its immediate neighbors. The attention matrix $A$ is then defined as:

$A=\text{softmax}\left(\frac{QK^{T}}{\sqrt{d_{k}}}M\right)$

where $Q$ and $K$ are the query and key matrices, $d_{k}$ is the dimension of the keys, and $M$ is the attention mask. The output of the attention mechanism is then:

$\text{Attention}(Q,K,V)=\text{softmax}\left(\frac{QK^{T}}{\sqrt{d_{k}}}M\right)V$

where $V$ is the value matrix. The computational complexity of this mechanism is $O(kw^{2})$, which is significantly lower than the $O(n^{2})$ of traditional attention when $w$ is much smaller than $n$.
