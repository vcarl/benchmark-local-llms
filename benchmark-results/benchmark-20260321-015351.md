def caesar_cipher(text: str, shift: int) -> str:
    def shift_char(c, shift):
        if c.islower():
            return chr((ord(c) - ord('a') + shift) % 26 + ord('a'))
        elif c.isupper():

### Approach
The approach chosen here is an iterative method to compute the Fibonacci number. This method is efficient in terms of both time and space complexity compared to the naive recursive approach, which has exponential time complexity. 

The iterative method works as follows:
1. **Base Cases**: If `n` is 0, return 0. If `n` is 1, return 1.
2. **Iterative Calculation**: For values of `n` greater than 1, initialize two variables to represent the first two Fibonacci numbers. Then, iterate from 2 to `n`, updating these variables to hold the last two Fibonacci numbers at each step.

This approach ensures that each Fibonacci number is computed in linear time, O(n), with constant space, O(1), as we only use a few variables to keep track of the sequence.

### Solution Code
```python
def fibonacci(n: int) -> int:
    """Return the nth Fibonacci number.
    
    fibonacci(0) = 0
    fibonacci(1) = 1
    fibonacci(n) = fibonacci(n-1) + fibonacci(n-2)
    """
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        a, b = 0, 1
        for i in range(2, n + 1):
            c = a + b
            a = b
            b = c
        return b
```

### Explanation
- **Base Cases Handling**: The function first checks if `n` is 0 or 1 and returns the corresponding Fibonacci number directly.
- **Initialization**: Two variables `a` and `b` are initialized to 0 and 1, representing the first two Fibonacci numbers.
- **Iterative Calculation**: Using a loop starting from 2 up to `n`, the function computes each subsequent Fibonacci number by summing the previous two values. The variables `a` and `b` are updated at each step to hold the last two computed Fibonacci numbers.
- **Efficiency**: This method efficiently computes the nth Fibonacci number in linear time with constant space, making it suitable for large values of `n`.

def word_frequency(text: str) -> dict[str, int]:

from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.cache = OrderedDict()
        self.capacity = capacity

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache[key] = value
            self.cache.move_to_end(key)
            return
        if len(self.cache) >= self.capacity:
            first_key = next(iter(self.cache))
            del self.cache[first_key]
        self.cache[key] = value
