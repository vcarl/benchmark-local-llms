This approach ensures that each Fibonacci number is computed in linear time, O(n), with constant space complexity, O(1).

### Solution Code
```python
def fibonacci(n: int) -> int:
    """Return the nth Fibonacci number.

    fibonacci(0) = 0
    fibonacci(1) = 1
```

class LRUCache:
    def __init__(self, capacity: int):
